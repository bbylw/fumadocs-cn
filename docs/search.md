# 搜索

在你的文档中实现文档搜索。

## 搜索 UI

参见 [搜索 UI](https://fumadocs.dev/docs/ui/search) 了解如何自定义文档搜索的 UI。

## 搜索客户端

你可以根据你的搜索引擎选择并配置搜索客户端，默认使用 Orama 搜索。

## 社区集成

由社区维护的集成列表：

- [Trieve Search](https://fumadocs.dev/docs/headless/search/trieve)
- [Typesense Search](https://fumadocs.dev/docs/headless/search/typesense)
